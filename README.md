# Full-Stack
Built a full-stack web application using React (Frontend), Spring Boot (Backend), and MySQL (Database) as part of a study-oriented learning project to strengthen full-stack development skills. Designed and implemented RESTful APIs to perform complete CRUD operations (Create, Read, Update, Delete), ensuring proper data flow and system reliability. Thoroughly tested all endpoints using Postman for accuracy and performance. The project emphasizes clean architecture, modular code design, and seamless communication between the frontend and backend to replicate real-world application standards.
